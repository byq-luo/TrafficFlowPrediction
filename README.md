# Traffic Flow Prediction
Traffic Flow Prediction using LSTM, GRU and RNN.

## Requirement
- Python 3.6    
- Tensorflow-gpu 1.14.0  
- Keras 2.2.5
- pandas 0.25.1
- numpy 1.16.4
- matplotlib 3.1.0
- scikit-learn 0.21.2

## Experiment

**Run the program:**

```
python main.py
```
**Print loss:**

```
python print_loss.py
```



## Train models

**Train models:**

```
python train.py
```
